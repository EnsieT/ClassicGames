Rules & Controls

RULES

  • Find all hidden words in the letter grid
  • Words may run horizontally, vertically, or diagonally
  • Some words can be backwards

CONTROLS

  • Drag across the grid from the first letter to the last
  • Mouse and touch both work
  • Uses the same Wordle word list source as the rest of this project
